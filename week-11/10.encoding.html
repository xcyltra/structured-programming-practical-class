Encoding & Decoding

📚 LEVEL 2 - MENENGAH (Penting)
Perbedaan encodeURI vs encodeURIComponent:
  • encodeURI() - Untuk encode URL lengkap, tidak encode : / ? # [ ] @
  • encodeURIComponent() - Untuk encode query parameter, encode semua karakter special
  • KESALAHAN UMUM: Pakai encodeURI() untuk query param (spasi tidak di-encode!)
  • BENAR: Selalu pakai encodeURIComponent() untuk query parameter!
Kapan Pakai Apa?
  • btoa/atob - Base64 encoding (simple data, bukan security!)
  • encodeURIComponent - Query string parameters (paling sering dipakai)
  • encodeURI - Full URL dengan protocol dan path
