Proxy & Reflect

📚 LEVEL 3 - ADVANCED (Opsional)
Metaprogramming:
  • Proxy memungkinkan intercept operasi pada object (get, set, delete, dll)
  • Reflect adalah built-in object untuk manipulasi object secara programmatic
  • Dipakai oleh framework seperti Vue.js untuk reactive data
  • Konsep advanced, fokus di materi dasar dulu sebelum mendalami ini
