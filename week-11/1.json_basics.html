JSON (JavaScript Object Notation)

📚 LEVEL 1 - DASAR (Wajib Dipelajari)
PENTING untuk Mahasiswa:
  • JSON TIDAK support: function, undefined, Symbol
  • Selalu gunakan try-catch untuk JSON.parse() karena bisa error
  • Circular reference akan menyebabkan error saat stringify
  • JSON hanya untuk data, bukan untuk method/function
Tips: JSON paling sering dipakai untuk API communication. Kalau kirim data ke server atau terima dari server, hampir pasti pakai JSON!
