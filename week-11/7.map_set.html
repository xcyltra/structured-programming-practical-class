Map & Set

📚 LEVEL 2 - MENENGAH (Penting)
🚨 KESALAHAN UMUM:

Map/Set pakai .size, bukan .length!

  • Array pakai .length
  • Map & Set pakai .size
  • map.length akan return undefined!
  • Set juga pakai .has() bukan .includes()
Kapan Pakai Map vs Object?
  • Gunakan Map jika: key bukan string, perlu iterasi berurutan, sering add/delete
  • Gunakan Object jika: key pasti string, perlu JSON.stringify(), struktur tetap
  • Set sempurna untuk: unique values, membership check, remove duplicates
