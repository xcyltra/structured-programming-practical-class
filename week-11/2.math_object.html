Math Object

📚 LEVEL 1 - DASAR (Wajib Dipelajari)
KESALAHAN UMUM dengan Math.random():
  • Math.random() return 0.0 - 0.999... (ada desimal!)
  • Untuk integer 0-9: Math.floor(Math.random() * 10)
  • Untuk integer 1-6 (dadu): Math.floor(Math.random() * 6) + 1
  • Rumus umum [min, max]: Math.floor(Math.random() * (max - min + 1)) + min
Tips Pembulatan:
  • Math.round() - Pembulatan normal (0.5 ke atas)
  • Math.ceil() - Selalu ke atas (ceiling)
  • Math.floor() - Selalu ke bawah (floor)
  • Math.trunc() - Buang desimal tanpa pembulatan
  • Untuk uang: (Math.round(harga * 100) / 100).toFixed(2)
