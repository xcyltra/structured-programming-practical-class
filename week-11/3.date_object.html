Date Object

📚 LEVEL 1 - DASAR (Wajib Dipelajari)
🚨 KESALAHAN PALING UMUM:

Bulan di JavaScript mulai dari 0, BUKAN 1!

  • Januari = 0, Februari = 1, ..., Desember = 11
  • new Date(2024, 1, 15) = 15 FEBRUARI, bukan 15 Januari!
  • new Date(2024, 12, 15) = Invalid Date! (maksimal 11)
  • Ini kesalahan yang paling sering dilakukan mahasiswa!
Tips:
  • Untuk timestamp, gunakan Date.now() atau getTime()
  • Format tanggal pakai toLocaleDateString('id-ID') untuk format Indonesia
  • Hitung selisih hari: (date2 - date1) / (1000 * 60 * 60 * 24)
