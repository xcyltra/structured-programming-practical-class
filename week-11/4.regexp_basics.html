Regular Expression (RegExp)

📚 LEVEL 2 - MENENGAH (Penting)
KESALAHAN UMUM RegExp:
  • Lupa escape karakter khusus: . * + ? ^ $ { } [ ] \ | ( )
  • Contoh: /gmail.com/ match "gmail_com" juga! Harus: /gmail\.com/
  • Lupa flag 'g' untuk replace all: replace(/hello/g, 'hi')
  • Flag 'i' untuk case-insensitive: /hello/i match "Hello", "HELLO"
Tips Belajar RegExp:
  • Gunakan regex101.com untuk test pattern
  • Mulai dari pattern sederhana, kompleksitas bertahap
  • Untuk email, gunakan library validasi (regex email sempurna sangat kompleks!)
  • Pattern umum: \d (digit), \w (word), \s (space)
