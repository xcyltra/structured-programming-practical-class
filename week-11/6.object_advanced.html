Object Advanced Methods

📚 LEVEL 2 - MENENGAH (Penting)
KESALAHAN UMUM:

Object.freeze() dan Object.assign() adalah SHALLOW!

  • Object.freeze() hanya freeze level pertama, nested object masih bisa diubah
  • Object.assign() hanya copy shallow, nested object masih reference yang sama
  • Untuk deep freeze/copy, perlu custom function atau library
Tips:
  • Gunakan Object.keys() untuk iterasi property object
  • Object.assign({}, obj) untuk shallow copy
  • Spread operator {...obj} lebih modern dan readable
  • Object.freeze() bagus untuk konstanta configuration
